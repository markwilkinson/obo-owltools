Preparation
-----------

Install Protege (4.2, but 4.1 may be fine)

  http://protege.stanford.edu

Install Elk (0.3 or higher)

  http://code.google.com/p/elk-reasoner/downloads/list

Install lego plugin

  http://code.google.com/p/owltools/downloads/list

  (note: if you are updating you need to remove the old one)

In Protege menu, select:

  Views/Ontology Views/Lego annotations

  (I place this in my Individuals tab)

OR

  Tabs/Lego annotations

place this where you like. I have it 

Getting started
---------------

  cd examples/
  cp template-worm.owl my-pathway-named.owl

open my-pathway-named.owl
